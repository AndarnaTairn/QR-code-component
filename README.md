### What I Learned

This project gave me hands-on experience with building a webpage from a design mockup. I improved my understanding of semantic HTML, Flexbox for layout, spacing, typography, and creating clean, maintainable CSS.

One technique I practiced was centering content using Flexbox:

css
body {
  display: flex;
  justify-content: center;
  align-items: center;
 }


### Continued Development

Going forward, I want to strengthen my CSS skills, especially responsive layouts, accessibility, and writing more scalable, reusable styles. I also plan to complete more Frontend Mentor challenges to improve my frontend development skills.
